摆苹果

小张有一篮大小不一的苹果，需要将其按大小摆放

已知每一个苹果有这两个属性：

* 1、苹果的序号
* 2、苹果的大小
* 构造一个数据结构：
    struct Apple  
    {
        //TODO your working
    }

要求：

* 1、将每一个苹果使用单链表的形式，将这些苹果链接起来。
* 2、根据每一个苹果的大小，按照降序进行排列。
* 3、在代码中实现一个函数：
    * Apple* ArrangeApples(Apple *appleList);
    *//选择一个自己认为时间复杂度最小的算法
    *//请勿修改函数头
    *//return 为一个排好序的苹果序列

Input

1 / 3
2 / 6
3 / 7
4 / 1
5 / 11
-1
* 1、第一个值为苹果的序号(num)
* 2、第二个值为苹果的大小(size)
* 3、输入-1表示结束输入
* 4、使用如上的格式输入

Output

5 | 11 -> 3 / 7 -> 2 | 6 -> 1 / 3 -> 4 / 1
*按照上面的格式输入摆好的苹果序列
